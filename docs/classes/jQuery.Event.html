resizestop リサイズ完了イベント

Defined in C:\xampp\htdocs\PROJECT\ampjs\develop\src\amp\jquery.plugins\jquery.event.stop.js:18

Event Payload:

  • 間引く時間を指定 Object

    default {timer: 200}

Example:

$(window).on('resizestop', callback);
$(window).on('resizestop', {timer: 1000}, callback);
