AMP.$

AMP.$ネームスペースは、jQuery用のプラグインクラスをエクスポートしています。
このクラスを使用する場合は、distフォルダ内にある以下のファイルをインストールしておいてください。

dist/
	lib/
		jquery-1.8.3.min.js (1.8.3以上推奨)
		jquery.plugins.js
		utilities.pack.js

	amp/
		amp.js
		jquery.plugins.js
		amp.jquery.js

		jquery.utilities/ (必要に応じてインストールしてください)
			Slider.js
			Float3d.js

Params

Arguments

Param Index Param Type Description
1 $boxHover jQuery 対象のbox要素
※省略可 初期値 $('.box-hover')
2 options Object オプション値
※省略可 初期値以下参照

Options

Option Key Type Default Description
hoverClass String hover ホバー時に、box要素に付与するクラス名
linkClass String link Box内に複数リンクがある場合、優先対象に指定するリンククラス名
